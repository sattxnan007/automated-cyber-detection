# "ผมได้พัฒนาโปรเจกต์ Automated Cyber Attack Detection โดยเริ่มจากการจำลองระบบที่มีช่องโหว่ด้วย Docker และใช้เครื่องมืออย่าง sqlmap จำลองพฤติกรรมการโจมตีจริงเพื่อเก็บข้อมูล Log. หลังจากนั้นผมได้นำข้อมูลดิบมาสกัดคุณลักษณะ (Feature Engineering) และสร้างโมเดล Machine Learning ใน Google Colab เพื่อตรวจจับความผิดปกติ. สิ่งที่ผมได้เรียนรู้อย่างมากคือความท้าทายในเรื่องข้อมูลที่จำกัดและความสำคัญของการวิเคราะห์ False Positive ในงาน Security จริงครับ"
